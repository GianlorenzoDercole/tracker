# Installation
+ to download django run pip3 install django==2.2
+ fork and clone this repo
+ cd into mysite and run python3 manage.py migrate
+ npm init -y
+ to install tailwind css run npm install tailwindcss@2.2.16

# Technologies used
+ Python
